## 2. FLUENT BUILDERS (Twig.TestKit)

### 2.1 WorkItemBuilder
Fluent builder for creating WorkItem test fixtures with sensible defaults.

Key methods:
- AsType()/AsTask/AsUserStory/AsFeature/AsEpic/AsBug/AsProductBacklogItem/AsRequirement/AsIssue
- InState(string state)
- WithParent(int parentId)
- AssignedTo(string? assignee)
- WithIterationPath(string path)
- WithAreaPath(string path)
- AsSeed(int daysOld = 0) - creates seed item
- LastSyncedAt(DateTimeOffset? value)
- Dirty() - marks item as dirty
- WithField(string key, string? value)
- WithFields(IEnumerable<KeyValuePair<string, string?>>)
- Build() - returns WorkItem

Factory: WorkItemBuilder.Simple(int id, string title)

### 2.3 WorkspaceBuilder
Fluent builder for workspace read models.

Methods:
- WithContext(WorkItem? context)
- WithSprintItems(params WorkItem[] items)
- WithSeeds(params WorkItem[] seeds)
- WithHierarchy(SprintHierarchy? hierarchy)
- Build() - returns Workspace

### 2.4 SprintHierarchyBuilder
Fluent builder for sprint hierarchy with parent lookups.

Methods:
- WithSprintItems(params WorkItem[] items)
- WithParents(params WorkItem[] parents)
- WithCeilingTypes(params string[] typeNames)
- WithTypeLevelMap(IReadOnlyDictionary<string, int> map)
- Build() - returns SprintHierarchy
